## Fantasy Racing Wireframe

Low-fidelity, mobile-first wireframe prototype for a fantasy horse racing app.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

Key routes:

- `/` Landing
- `/login` Entry screen
- `/league` League home
- `/draft` Draft room
- `/team` My team (start/sit)
- `/matchup` Weekly matchup
- `/standings` League standings
